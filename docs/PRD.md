# 💎 PRD: Didim V3.0 상세 기능 명세

## 1. 핵심 기능: "The Ministry Map Engine"
사용자의 프로필을 분석하여 9개 사업 중 최적의 지원 경로를 도출합니다.

### 1.1 로직 우선순위 (Hierarchy)
1.  **고용노동부 (근로자):** 한도 높음, 자부담 없음 → **1순위 추천**
2.  **국가보훈부 (유공자):** 국비 100% → **1순위 추천**
3.  **교육부 (학생):** 학습용 기기 → **학교 신청 가이드**
4.  **과기정통부:** 5~6월 시즌성 → **사전 예약 알림**
5.  **보건복지부 (장애인/노인):** 가장 보편적 → **기본 베이스**

## 2. 기능 상세 요구사항

### A. 디딤 체크 (Integrated Assessment)
*   **Input:** 나이, 장애유형/정도, **사회적 신분(학생/직장인/구직자)**, 소득분위(자부담율 계산용).
*   **Output:** "홍길동님은 [고용노동부 보조공학기기 지원사업] 대상자입니다. 예상 자부담금은 0원입니다."

### B. 서류 자동 생성기 (Auto-Form Generator)
*   **기능:** "신청서 작성하기" 버튼 클릭 시.
*   **AI Writing:** Gemini가 사용자의 장애 상황과 직무(또는 생활) 환경을 결합하여 **'기기 필요성 및 활용 계획서'** 자동 작문.
*   **File Export:** 데이터가 기입된 HWP/PDF 파일 즉시 다운로드.

### C. n8n Auto-Sourcing
*   **Schedule:** 매주 월요일 09:00.
*   **Sources:** 식약처 의료기기 DB, 공공데이터포털, 네이버 쇼핑.
*   **Action:** 신규 제품 발견 -> Gemini가 '급여 매핑' 분석 -> DB 임시 저장 -> 관리자 승인.

## 3. 사용자 플로우 (User Flow)
1.  **접속 & 로그인:** 카카오톡으로 3초 가입.
2.  **상황 진단:** 채팅형 인터페이스로 정보 입력 (서류 용어 사용 X, 일상 용어 사용 O).
3.  **결과 확인:** 추천 제품 + **[국비 지원 최적 경로]** 제시.
4.  **서류 준비:** AI가 써준 신청서 확인 및 다운로드.
5.  **신청/구매:** 온라인 신청 링크 이동 또는 제휴 사업소 연결.
